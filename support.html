Support

Last Updated: June 15, 2026

1. Welcome to Brofolio Support

Thank you for using Brofolio. We are committed to helping real estate professionals manage their portfolios efficiently and effectively.

If you experience any issues, have questions about app functionality, subscriptions, data management, or would like to provide feedback, we are here to assist.

2. How Brofolio Works

Brofolio is a portfolio management application designed specifically for real estate agents and property professionals.

The app allows users to:

  • Create and manage real estate portfolios
  • Store property details and characteristics
  • Create and organize customer contact lists
  • Record customer property requirements and preferences
  • Match available properties with customer search criteria
  • Import and export data using CSV files
  • Manage subscriptions and premium features

3. Common Issues and Solutions

  • Data Not Appearing: Ensure you are connected to the internet and reopen the application.
  • Portfolio Matching Not Working: Verify that both property information and customer search criteria have been entered completely.
  • CSV Import Failed: Confirm that the CSV file follows the required format and contains valid data.
  • CSV Export Not Available: Ensure the application has permission to access and save files on your device.
  • App Performance Issues: Restart the application and make sure you are using the latest version available.
  • Subscription Not Recognized: Use the "Restore Purchases" option inside the app to synchronize your subscription status.

4. Subscriptions and Purchases

Brofolio offers optional premium features through subscriptions.

All purchases are securely processed through Apple In-App Purchases and managed through RevenueCat.

If your subscription is active but premium features are unavailable, please use the "Restore Purchases" option within the application.

If the issue persists, contact support with your User ID and a description of the problem.

5. CSV Import and Export Support

Brofolio supports CSV import and export functionality to simplify portfolio and customer data management.

If you encounter issues importing CSV files, please verify:

  • The file is in CSV format (.csv)
  • Column headers are correctly formatted
  • Required fields contain valid values
  • The file is not corrupted or incomplete

For assistance with CSV-related issues, please include a description of the error when contacting support.

8. Troubleshooting Tips

  • Keep Brofolio updated to the latest version.
  • Ensure your internet connection is stable.
  • Restart the application if unexpected behavior occurs.
  • Verify that imported CSV files use the correct format.
  • Use "Restore Purchases" if premium features are missing.
